# nickel-tera

[Nickel](http://nickel.rs/) Request Extension for [Tera](https://github.com/Keats/tera)

## Installation

Cargo.toml
```
[dependencies.nickel_tera]
git = "https://github.com/hhatto/nickel-tera.git"
```
